Это webview интерфейс пакета wordlexord.
Пронумеруем все слова в выбранном алфавите (например английском):
a -> 1, b -> 2, ..., x -> 24, y -> 25, z -> 26,
aa -> 27, ab -> 28, ac -> 29, ...

Тогда у всякого слова будет свой порядковый номер.

В стандартном английском алфавите "abcdefghijklmnopqrstuvwxyz" 8514ое слово - lol.
Так же можно получить слово по его порядковому номеру. Порядковый номер слова jesus - 4671049.
#1. Алфавиты
Сейчас доступно 4 алфавита:
рул - русский ловеркейс: абвгдеёжзийклмнопрстуфхцчшщъыьэюя
руу - русский апперкейс АБВГДЕЁЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ
enl - английский ловеркейс: abcdefghijklmnopqrstuvwxyz
enu - английский апперкейс: ABCDEFGHIJKLMNOPQRSTUVWXYZ

В пакете можно установить любой алфавит.

#2. Тулзы
В пакете доступны разные методы для работы со словами и числами. Подробнее в README

Основной функционал webview:

[alphabet] [text] - Если text - набор порядковых номеров в алфавите alphabet, то переводит их в слова, и наоборот, переводит слова с порядковые номера.

Некоторые нюансы:

Если передать текст, не состоящий из чисел, то в нём удалятся все символы, отсутствующие в алфавите. Например, если передать руу привет, то ничего не вернётся, т.к. в слове привет нет ни одной заглавной буквы. Если же передать руу Привет!, то вернётся лишь порядковый номер буквы П, т.к. остальные отсутствуют в алфавите из заглавных букв.
